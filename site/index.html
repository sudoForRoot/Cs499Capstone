Introduction

Professional Self-Assessment

Completing the B.S. in Computer Science at SNHU has been a process of building capability layer by layer — from foundational programming through object-oriented design, data structures, software testing, and into this capstone experience that required synthesizing all of it into a coherent, professional body of work.

All three enhancement categories were applied to the same artifact: the Grazioso Salvare Rescue Animal Dashboard, originally developed in CS 340. This deliberate choice demonstrates not only competency in each area individually, but the ability to reason about an entire software system and make targeted, justified improvements without breaking the whole.

The skills developed across this program — collaborating in team environments, communicating with stakeholders, applying algorithmic thinking, building secure systems, and managing database resources — are demonstrated throughout the artifacts that follow.
